## 📖 Deskripsi Project

Project ini merupakan pengembangan fitur **Shopping Cart berbasis web** 
Aplikasi ini dibuat untuk memenuhi tugas *Shopping Cart Enhancement* dengan mengimplementasikan seluruh fitur wajib serta fitur bonus yang telah ditentukan.

Aplikasi memungkinkan pengguna untuk:

- Menambahkan produk ke dalam keranjang (Add to Cart)
- Melihat daftar item dalam keranjang beserta jumlah (quantity)
- Menambah dan mengurangi jumlah produk (+ / -)
- Menghapus produk dari keranjang
- Melihat total harga yang terhitung otomatis
- Mencari produk berdasarkan nama (Search)
- Memfilter produk berdasarkan kategori (Category Filter)
- Melakukan proses checkout dengan ringkasan pesanan (Order Summary) dan form data pelanggan

Data keranjang disimpan sementara menggunakan **LocalStorage**, sehingga dapat diteruskan ke halaman checkout.
