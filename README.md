## 📖 Deskripsi Project

Project ini merupakan pengembangan fitur **Shopping Cart berbasis web** yang dibuat untuk memenuhi tugas *Shopping Cart Enhancement*.

Aplikasi ini mensimulasikan sistem keranjang belanja pada e-commerce, di mana pengguna dapat:

- Menambahkan produk ke keranjang
- Melihat daftar item dalam keranjang
- Mengatur jumlah pembelian
- Menghapus produk
- Melihat total harga otomatis
- Mencari produk berdasarkan nama
- Memfilter produk berdasarkan kategori
- Melakukan proses checkout

Data keranjang disimpan sementara menggunakan **LocalStorage** sehingga tetap tersimpan meskipun halaman di-refresh.

---

## 🎯 Fitur Utama (Wajib)

### ✅ 1. Add to Cart
Pengguna dapat menambahkan produk ke dalam keranjang dengan menekan tombol **Add to Cart**.

### ✅ 2. Menampilkan Item di Keranjang
Produk yang telah ditambahkan akan ditampilkan lengkap dengan:
- Nama produk
- Harga produk
- Quantity (jumlah)
- Subtotal

### ✅ 3. Update Quantity (+ / -)
Pengguna dapat:
- Menambah jumlah produk (+)
- Mengurangi jumlah produk (-)

Total harga akan otomatis diperbarui sesuai perubahan jumlah.

### ✅ 4. Remove Item
Produk dapat dihapus dari keranjang menggunakan tombol **Remove**.

### ✅ 5. Total Harga Otomatis
Total harga dihitung secara otomatis berdasarkan jumlah dan harga produk.
